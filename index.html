Studentlist

Visit website

The site is not well designed since there was not an implicit focus on the design but rather the functionality. The purpose of this project was to develop a student list with different features such as sorting, filtering, removing students from the list, adding them to the inquisitorial squad, etc. My main focus was implementing the features using JavaScript and more importantly to understand the logic behind programming with JavaScript. OBS! The site is a ‘hacked’ ... version, which is why it might seem like some of the functions do not work. I will recommend you to have a look at the code behind - or contact me so I can explain. Campaign

Visit website

The idea behind this campaign for the NGO Rådet for Sikker Trafik, is to bring awareness about the importance of being visible in the traffic while riding a bicycle. The target audience is elderly people who either daily or weekly travel by bike. This project was focused on data analysis and online communication to a specific target audience. This target group in particular might be having a harder time using computers, so this was a point of emphasis for the design.
